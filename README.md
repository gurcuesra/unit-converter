# This project is for the Unit Converter

It includes converters:

- Kelvin to Celsius
- Celsius to Fahrenheit
- Celsius to Newton
- MPH to KMPH

Formula of the Kelvin to Celsius:

- Celsius = Kelvin - 273;

Formula of the Celsius to Fahrenheit:

- Fahrenheit = Celsius \* (9/5) + 32;

Formula of the Celsius to Newton:

- Newton = Celsius \* (33 / 100);

Formula of the Miles per hour to Kilometers per hour:

- KMPH = MPH \* 1.6;

P.S: Fahrenheit temperature will be round. It will NOT display the decimal points!!!

P.S-2: Newton scale will be round the lower integer. It will NOT display the decimal points!!!
